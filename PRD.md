# PRD: Follow Me!

**1. 제품명:** **Follow Me!** - Follow me!

**2. 비전:** 모든 사용자가 어떤 웹사이트에서든 길을 잃지 않고, 마치 익숙한 길을 걷는 것처럼 편안하게 목표를 달성할 수 있도록 돕는다.

**3. 대상 고객:**
*   새롭거나 복잡한 구조의 웹사이트(예: 정부 사이트, 대규모 이커머스, SaaS 서비스)를 처음 방문하는 사용자
*   정보 탐색에 어려움을 느끼는 비숙련 인터넷 사용자

**4. 해결하려는 문제:**
*   사용자는 낯선 웹사이트에서 원하는 정보를 찾기 위해 어디를 클릭해야 할지 막막함을 느낀다.
*   이러한 불확실성은 사용자의 이탈률을 높이고, 서비스 경험을 저해한다.

**5. 핵심 기능 및 사용자 스토리:**
*   **사용자 행동 데이터 수집 (User Behavior Tracking):**
    *   *As a user,* 나의 클릭, 입력, 페이지 이동 등 웹 서핑 행동은 개인 식별 정보가 제거된 상태로 안전하게 수집되어 서비스 개선에 사용된다.
*   **성공 경로 분석 및 가이드 제공 (Success Path Analysis & Guidance):**
    *   *As a user,* 특정 목표(예: 회원가입)를 달성한 다른 사용자들의 행동 패턴을 기반으로, 나에게 가장 적합한 단계별 가이드를 인터랙티브하게 제공받는다.
*   **데이터 수집 동의 및 제어 (Consent & Control):**
    *   *As a user,* 나는 확장 프로그램을 처음 실행할 때 어떤 데이터가 수집되는지 명확히 안내받고 동의/거부를 선택할 수 있다.
    *   *As a user,* 나는 원할 때 언제든지 설정 메뉴에서 데이터 수집 기능을 켜고 끌 수 있다.

**6. 성공 지표:**
*   **활성화율:** 설치한 사용자 중 추천 기능을 활성화한 사용자의 비율
*   **가이드라인 상호작용률:** 제공된 가이드라인을 따라 행동을 완료한 사용자의 비율
*   **사용자 유지율(Retention):** 한 달 후에도 확장 프로그램을 계속 사용하는 사용자의 비율
*   **목표 전환율 개선:** 가이드라인을 제공받은 사용자가 그렇지 않은 사용자보다 목표(예: 구매, 가입)를 더 높은 비율로 달성하는가?

**7. 1차 버전 제외 범위 (Out of Scope for V1):**
*   실패 경로 분석 및 시각화 (Failure Path Analysis & Visualization)
*   개인화된 사용자 계정 시스템
*   A/B 테스트를 통한 시각화 방법 자동 최적화
